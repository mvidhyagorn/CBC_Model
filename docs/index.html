SEIR difference equations — Table S5

Discretized SEIR model. Each day (t) at stage (p) computed via difference equations. All equations from Additional File 2, Table S5.

Initial conditions (t = 0):
S₀ = N − α·σ·N   E₀ = ½·α·σ·N   I₀ = ½·α·σ·N   R₀ = 0
N = 100,000 travelers/week. α = cluster infection rate/million/day. σ = 1 − ϑ·ω (vaccine-adjusted transmission). Initial infected split 50:50 between E and I compartments.
Susceptible compartment:
S(t,p) = S(t−1,p−1) − α·σ · S(t−1,p−1)
Fraction α·σ of susceptibles transitions to exposed each day. During quarantine stages, α = 0 (no new transmission). During travel (P4), α is doubled for close-contact risk. [26]
Exposed compartment:
E(t,p) = E(t−1,p−1) + α·σ·S(t−1,p−1) − β·E(t−1,p−1) − (1−μ)·E(t−1,p−1)
Gains: new infections from S. Losses: β = 1/incubation transitions to I [4]; testing detection (1−μ) where μ = 0.84 false-negative rate during latency [1]. Testing term applies only at testing stages (P2, A).
Infectious compartment:
I(t,p) = I(t−1,p−1) + β·E(t−1,p−1) − γ·I(t−1,p−1) − (1−τ)·I(t−1,p−1) − π·(1−ρ)·I(t−1,p−1)
Gains: E→I at rate β. Losses: recovery γ = 1/14 [5]; testing detection (1−τ) where τ = 0.29 [1]; severe symptoms π·(1−ρ) cause trip cancellation or detection [6].
Recovered:
R(t,p) = R(t−1,p−1) + γ · I(t−1,p−1)
Accumulates at recovery rate γ = 1/infectious period.
Missed cases (primary output):
MC(t,p) = E(t,p) + I(t,p)
Undetected infected travelers at any stage. Final MC after all stages = individuals entering Thai communities undetected.
Vaccine reduction factors:
σ = 1 − ϑ·ω  (transmission)  ρ = 1 − δ·ω  (severity)
ϑ = 0.30 vaccine efficacy vs transmission [2]. δ = 0.70 vs severe disease [3]. ω = vaccination coverage (cluster-specific, or 1.0 if certificate required).
